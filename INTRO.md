## 주제 : 34. 미세먼지 데이터 시각화

### 주제선정 배경
* 공공데이터 등을 이용하여 우리나라 최근 5~10년 사이의 미세먼지 변화 흐름을 시각화 한다.

### 개발 요구사항 (필수)
* Full-Stack 개발환경  
  - API 활용능력
  - 데이터 저장 & 가공능력 
  - 시각화 라이브러리 사용능력 (D3 등)
  - 네이버 지도 등 지도 API 사용능력

### 개발 요구사항 (선택)
- Backend or Frontend(JS) 개발 능력

### Role
* Full-stack Developer
* Back-end Developer
* Front-end Developer

### Technology 
- Backend : Spring or  Django or Node.js
- Frontend : JS 기반

### 개발언어
* Java
* Python
* JavaScript

### 기타사항
* 공공데이터 활용
